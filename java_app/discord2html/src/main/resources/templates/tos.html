運用ポリシー（Discordアーカイブ公開）

最終更新日: 2025-09-26

本Bot/アーカイブシステムについて

当コミュニティのDiscordサーバにおける発言をアーカイブし、外部に公開するに際し、その方針・手続・利用者の権利を示します。

  • 開発者情報: GitHub owner
  • ソースコード: GitHub公開レポジトリ owner/repo
  • 運用: DiscordBotとGitHubActionによる自動運用。24時間常時待機ですが、技術的問題で一時的に停止する場合があります。

アーカイブの対象・範囲

  • 対象チャンネル: 当コミュニティ内に設置されているすべてのチャンネル・スレッド。一部除外チャンネルあり
  • 対象コンテンツ: テキスト、埋め込み、添付（画像/ファイル/コード）、リアクション、返信/引用のメタ情報
  • 対象期間: アーカイブ作成時点から遡って概ね24時間分を取得します。

匿名化ポリシーとロールによる例外

  • 基本方針: ユーザ識別子は匿名化し、管理者が指定する時間範囲とチャンネル・スレッド単位で一時IDを付与します
  • ロール例外: 特定ロール（管理者・運営など）は、匿名化されない場合があります
  • 公開可否: 匿名化対象ロールであっても、本人の明示同意により非匿名で公開可能です
  • 匿名化されない情報: テキスト内でメンションにされず言及されたID/アカウント名/その他呼称は匿名化されず公開されます

オプトアウト（不開示希望）

以下の方法で不開示希望を登録・確認できます。

  • コマンド例: /d2h optout opt-in:False
  • ヘルプ: /d2h help

登録は即時反映を試みますが、技術的失敗時は公開を保留します。

技術的問題により不開示の希望状態を喪失した場合には速やかに当Discordコミュニティ内で告知します。 その際は改めて不開示希望の旨を再登録いただくようご協力をお願いいたします。

記録・削除・再生成

  • 取得対象は最大「アーカイブ作成時点の前7日」です。通常、日本標準時深夜0時に起算し前24時間を取得します
  • アーカイブ作成以後にDiscord上でメッセージが削除されても、既に取得済みデータには自動反映されません
    • 削除対応を希望する場合、管理者にアーカイブ再生成をリクエストしてください（目安: 7日以内）
    • 再生成により公開ページは更新されますが、履歴管理の都合上、外部リポジトリ等に記録が残る場合があります
  • 発言の存在のみを保持/公開する場合があります
  • 当サーバー以外から転送されたメッセージについては、オプトアウトできないためアーカイブの対象外となります

個人情報・機微情報の取り扱い

  • 投稿内容（添付の画像などを含む）に記述された情報はアーカイブとして公開されます。 機微な個人情報を含まないよう、留意してください。

利用目的・保存期間

  • 目的: 技術的知見の共有、コミュニティ運営の透明性向上、議論の再利用性確保
  • 準拠規則: 日本国の法、Discordポリシーに準拠します
  • 保存期間: 目的に適する限り保持します。 なお、管理者が当アーカイブを非公開とした場合でも、インターネット上に公開された情報として第三者により複製保存される場合があります
  • 第三者提供: 公開サイト/リポジトリでの一般公開を含みます

利用者の権利

  • アクセス権・訂正権・削除権・処理制限/異議申立ての権利を尊重します
    • 技術的要件により公開された履歴として残る場合があります
  • 請求方法: 当コミュニティDiscordサーバの対応チャンネルからリクエストしてください
    • 注意事項: botあるいは管理者のDMでは対応いたしかねます
    • 技術的問題は当ポリシー冒頭記載の開発者と連携することがあります
  • 対応目安: 30日以内に対応（繁忙時は延長の可能性があります）

安全管理措置

  • オプトアウト情報や監査ログはアクセスを制御した環境に限り保存、リスクの制御に務めます
  • Discordポリシーに基づき、ユーザーは13歳以上などの利用条件と伴う責任とを確認します
    • 親権者など適切な者は当コミュニティにおいて対象のDiscordアカウントを経由して、 当ポリシーに同意すること、あるいは必要なオプトアウトをすることができます

透明性と変更

  • 本ポリシーの変更時は、サーバ内で通知し、必要に応じて対応すべき内容を通知します。
  • 本ページ（https://example.github.io/repo）にポリシー/生成手順/匿名化仕様の概要と問い合わせ窓口を明示します
